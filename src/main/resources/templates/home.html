Welcome to Cloud Native Buildpacks 101!

If you're seing this page, this means you successfully created a Docker image using the CLI pack from Cloud Native Buildpacks.

This app is powered by Spring Boot, running on Java .

What if I want to rebuild this image?

Here is a command you may use to build this image from source code:

$ pack build cloudnativebuildpacks101 --builder=nebhale/java-build

The source code will automatically be compiled, and a Docker image named cloudnativebuildpacks101 will be created, using the next generation of Cloud Foundry Java Buildpack, maintained by Ben Hale.

Run the app using this Docker command:

$ docker run --rm -p 8080:8080/tcp cloudnativebuildpacks101
